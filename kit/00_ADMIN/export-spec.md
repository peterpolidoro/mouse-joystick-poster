# Export / File Naming Spec

## Poster template
- Edit `01_INKSCAPE/mbm-comic-poster-template-unflattening-white.svg`
- Keep relative paths so the title bar image stays linked.

## Panel image outputs (recommended)
- Format: PNG (preferred) or TIFF
- Color: RGB while working; convert to CMYK only if your printer requires it.
- Resolution: 300 dpi (150 dpi minimum if needed)

## File naming
Save final panel images as:
- `P01.png` ... `P10.png`

Place them in:
- `03_PANELS/P01_*/output/P01.png`, etc.

## Suggested pixel dimensions (at 300 dpi)
These match the *mask sizes* in the template (approx; feel free to go larger):

- P01: 9.4×10.9 in → ~2820×3270 px
- P02: 26.0×12.0 in → ~7800×3600 px
- P03: 7.9×10.7 in → ~2370×3210 px
- P04: 27.1×16.1 in → ~8130×4830 px
- P05: 15.6×15.1 in → ~4680×4530 px
- P06: 14.4×7.1 in → ~4320×2130 px
- P07: 17.2×8.5 in → ~5160×2550 px
- P08: 9.9×7.1 in → ~2970×2130 px
- P09: 8.1×8.1 in → ~2430×2430 px
- P10: 43.5×5.5 in → ~13050×1650 px

## Inkscape insertion rule of thumb
Generate with a bit of extra margin; the clip masks will crop.
